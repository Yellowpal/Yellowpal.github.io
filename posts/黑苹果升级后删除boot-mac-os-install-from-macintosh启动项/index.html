黑苹果升级后删除Boot MAC OS Install From Macintosh启动项

2020.04.03

最近黑苹果升级10.15.4的时候中间提示失败了，但是重新进入系统是提示升级成功了，不过启动引导残留了下来了个Boot MAC OS Install from Macintosh的选项，最要命的是还是默认启动项，网上搜索一番无果后自己查看了下本地目录的文件，最终找到了需要删除的文件在这个目录下/System/Volumes/Data，删除命令如下：

sudo rm -rf /System/Volumes/Data/macOS\ Install\ Data

之后重启一切正常，残留的引导项成功被删了。
